Features

The Markdownify MCP server can convert the following file types to Markdown:

  • PDF files
  • Images (with metadata and description)
  • Audio files (with transcription)
  • DOCX files
  • XLSX files
  • PPTX files
  • YouTube videos (with transcript)
  • Bing search results
  • Web pages (like this one)

Code Example

Here's an example of how to use the Markdownify MCP server in JavaScript:

// Example code to use the Markdownify MCP server
const markdownify = require('markdownify-mcp');

async function convertToMarkdown(url) {
    try {
        const result = await markdownify.webpageToMarkdown(url);
        console.log(result);
    } catch (error) {
        console.error('Error converting webpage to Markdown:', error);
    }
}

convertToMarkdown('https://example.com');
